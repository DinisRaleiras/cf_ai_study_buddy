# PROMPTS.md

This file documents how AI tools were used during the development of this project.

## System Prompt (Application)

The application uses the following system prompt:

"You are a helpful study assistant for a Computer Science student. Give clear, concise, and accurate answers."

This prompt is used to guide the model's behavior during conversations.

## Prompts Used During Development

During development, AI tools were used to support implementation, debugging, and structuring the project.

Examples of prompts used:

- "How can I build a simple AI chat app using Cloudflare Workers and Durable Objects?"
- "Explain how to store conversation memory using Durable Objects"
- "Generate a minimal frontend chat interface in HTML and JavaScript"
- "Help me structure a README for a Cloudflare AI project"
- "How do I connect a Worker to a Durable Object using session IDs?"
- "Debug why my Cloudflare Worker is not returning a response"

## Notes

AI was used as a support tool during development, mainly to:
- understand Cloudflare APIs
- structure the application
- improve clarity and documentation

All final implementation decisions and code adjustments were reviewed and adapted manually.
